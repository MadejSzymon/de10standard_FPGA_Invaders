# de10standard_FPGA_Invaders
Game created for a DE10 Standard FPGA Development Kit

Controls:

SW0 and SW1 - enable VGA

BTN3 - move left

BTN2 - move right

SW4 - pause

BTN1 - shoot


BTN0 - reset

SW9 - play (title screen)
